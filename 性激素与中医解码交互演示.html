E2

雌激素

长内膜，促卵泡成熟
维持女性第二性征

中医：肾阴与肝血

生理基础：滋润、充养胞宫的物质基础，月经前半段阴血渐充。

病理表现：若不足即为肝肾阴虚（经量少、干涩、潮热）；过高则易生瘀滞。

P

孕激素

升体温，稳内膜，保胎
为受精卵着床做准备

中医：肾阳与脾气

生理基础：体温升高是“肾阳”温煦；稳固对抗雌激素是“脾气”固摄统血。

病理表现：若不足即为脾肾阳虚（内膜不稳、经前漏血、易流产）。

FSH

促卵泡生成素

刺激卵巢卵泡发育
评估卵巢储备的关键

中医：肾精与肾气

生理基础：“肾精”化生天癸，“肾气”推动卵泡生长的生殖原动力。

病理表现：异常升高即为肾精亏虚、虚火上炎，底子空虚。

LH

促黄体生成素

排卵前的关键高峰
诱发排卵，促黄体形成

中医：重阴必阳的枢纽

生理基础：心气下达、肾阳鼓动，触发阴阳转化的瞬间（排卵）。

病理表现：排卵障碍多为心肾不交或气滞血瘀，扳机扣不动。

T

雄激素

提供精力与肌肉力量
过高会导致痤疮多毛

中医：命门火与相火

生理基础：正常为“命门之火”，给人活力与生机。

病理表现：过高为病理性“邪火”，即相火妄动、肝经郁热。

PRL

泌乳素

促进乳汁分泌
非哺乳期过高抑制排卵

中医：肝气与脾胃

生理基础：由脾胃气血化生，其分泌受“肝气”疏泄控制。

病理表现：高泌乳素血症核心为肝气郁结，横逆犯脾。

月经期 (行经期)

推旧生新
激素变化

雌激素 (E2) 和 孕激素 (P) 均处于整个周期的最低谷。激素撤退导致子宫内膜失去支撑，剥脱出血。

中医气血

阳气转化为阴气，气血下行汇聚于胞宫。此时以“通”为顺，旧血排空，新血才能生化。

🩺 调治原则：理气活血，重在排瘀。

卵泡期 (经后期)

阴血渐充
激素变化

FSH 轻微上升刺激卵泡发育；随着卵泡长大，雌激素 (E2) 稳步升高，促使子宫内膜重新生长增厚。

中医气血

经血排出后，子宫相对空虚。此时体内阴血渐充，肾阴逐渐旺盛，为下一次排卵积累“物质基础”。

🩺 调治原则：滋补肝肾，养血填精。

排卵期 (经间期)

重阴必阳
激素变化

雌激素 (E2) 达到极高值，正反馈触发 LH 激增形成峰值，诱发卵泡破裂排卵。

中医气血

阴气积累到极点瞬间产生质变，即“重阴必阳”。心肾相交，阳气萌动，这是阴阳交替转化的枢纽。

🩺 调治原则：温肾和血，促发排卵。

黄体期 (经前期)

阳气最盛
激素变化

排卵后形成黄体，孕激素 (P) 大量分泌并占据主导，基础体温升高；联合雌激素维持内膜稳定。

中医气血

整个周期中肾阳最旺盛的时期。阳气负责温煦胞宫，同时脾气发挥“固摄”作用，为受精卵着床做准备。

🩺 调治原则：温阳健脾，疏肝理气。

病理演变：妇科常见问题的双向解码

当周期节律与激素平衡被打破时，看看中西医是如何分别干预的

雄激素高 排卵障碍

📊 西医视角

核心特征是雄激素(T)偏高，或LH/FSH比值失调。卵泡多但长不大、排不出。常伴有胰岛素抵抗和肥胖。

☯️ 中医解码

  • 雄激素高：相火妄动 / 肝经郁热（表现为长痘、多毛、脾气急躁）。
  • 无LH峰排卵：肾虚底子不足 + 痰湿阻滞。痰湿像厚壳包裹卵巢，导致阴阳转换的“扳机”扣不动。

🌿 中医调理比喻：【疏通河道、清理淤泥】

思路：补肾化痰，清热平肝。用健脾化痰药（苍术、茯苓）剥除卵巢表面的“痰湿”；用滋阴降火药压制相火；排卵期加活血化瘀药助推排卵。

孕酮偏低 内膜不稳

📊 西医视角

排卵后黄体发育不良，导致孕激素(P)分泌不足。表现为基础体温上升慢、高温期短、经前漏血，或孕早期极易流产。

☯️ 中医解码

  • 孕酮低：脾肾阳虚，气不摄血。黄体期本应是“阳气最盛”之时。
  • 病机：“肾阳”不足体温不升；“脾气”虚弱统摄不住血液，导致内膜提前剥脱。

🌿 中医调理比喻：【添柴加火、加固堤坝】

思路：温补脾肾，固摄冲任。此时不仅不用清热疏肝，还要大量使用温热、固涩之药（如菟丝子、杜仲补肾阳；黄芪补脾气），自身托起阳气产生孕激素。

激素波动 泌乳素偏高

📊 西医视角

经前雌、孕激素急剧波动，或伴轻度泌乳素(PRL)升高，影响神经递质。表现为经前乳房胀痛、情绪暴躁、腹胀等。

☯️ 中医解码

  • 波动/泌乳素偏高：肝失疏泄，气机郁滞。
  • 病机：经前气血向下汇聚胞宫，导致上半身空虚且“肝气”容易拥堵。堵在乳房则胀痛，堵在头部则头痛，堵在脾胃则腹胀。

🌿 中医调理比喻：【调节交通、疏散拥堵】

思路：疏肝解郁，柔肝养血。代表方“逍遥散”。柴胡理顺郁结肝气，当归、白芍补充肝血柔和肝脏。交通顺畅，情绪与胀痛自然缓解。

雌激素无对抗 异常出血 (崩漏) 瘀血阻滞

📊 西医视角

通常因为无排卵，导致单一雌激素(E2)长期刺激子宫内膜，而缺乏孕激素(P)的对抗和转化。内膜异常增厚，脆弱剥脱，导致月经量过大、经期延长或不规则大出血（且有癌变风险）。

☯️ 中医解码

  • 雌多孕少：阴长而无阳化。即“物质”（阴/内膜）疯狂堆积，缺乏“阳气”（孕激素）的转化力量。
  • 病机本质：异常增厚的内膜在中医看来属于离经之血、病理产物，即“瘀血阻滞胞宫”。瘀血不去，新血不生，血不归经，故发为“崩漏”（崩为大出血，漏为滴答不尽）。

🌿 中医调理比喻：【破瘀生新、固本澄源】

思路：急则治标（化瘀止血），缓则治本（重建周期）。出血严重时，需用“活血化瘀”之药（如三七、蒲黄、三棱，犹如中医的“药物刮宫”）剥落坏死内膜，并配合益气固摄药止血；血止后，通过温阳滋阴重建正常的排卵轴。

📊 西医对症用药 (按需组合)

  • 降雄激素
    达英-35 (复方醋酸环丙孕酮)：临床最常用，抑制卵巢和肾上腺分泌雄激素。
  • 调糖脂
    二甲双胍：改善胰岛素抵抗，降低体重，辅助降低雄激素。
  • 促排卵
    来曲唑 (Letrozole) / 克罗米芬：针对有生育要求者，诱发排卵。
  • 调周期
    地屈孕酮 / 黄体酮：定期诱发撤退性出血（药物刮宫），保护子宫内膜不发生病变。

☯️ 中医人工周期疗法 (按期调治)

1. 卵泡期 (经后期)： 治法：滋补肝肾，化痰软坚。
代表方：苍附导痰丸 合 左归丸 加减。剥除卵巢痰湿，促卵泡发育。
2. 排卵期 (经间期)： 治法：活血化瘀，补肾促排。
代表方：促排卵汤 (含丹参、赤芍、皂角刺)。助力“重阴必阳”，冲破卵泡壁。
3. 黄体期 (经前期)： 治法：温补脾肾，扶助阳气。
代表方：右归丸 或 毓麟珠。提高黄体功能，升高基础体温。
4. 月经期 (行经期)： 治法：活血化瘀，理气调经。
代表方：桃红四物汤。促使经血排净，去旧生新。

📊 西医对症用药

  • 直接补充
    地屈孕酮 (达芙通) / 微粒化黄体酮胶囊：排卵后连续使用10-14天，最直接有效补充孕激素。
  • 黄体支持
    hCG注射液：排卵后肌肉注射，模拟LH功能，刺激自身的黄体发育并分泌孕酮。
  • 溯源促排
    克罗米芬 / 来曲唑：遵循“好卵泡才有好黄体”原则，通过改善前端卵泡质量来提升后端的黄体功能。

☯️ 中医人工周期疗法

重点时期：黄体期 (经前期) 治法：温补脾肾，固摄冲任。这阶段阳虚气弱是核心病机。
代表方：寿胎丸 (菟丝子、桑寄生、续断、阿胶) 合 四君子汤 (党参、白术等) 或 健固汤。
次重点期：卵泡期 (经后期) 治法：阴阳互根。中医讲究“善补阳者，必于阴中求阳”。卵泡期要养精血，打好物质基础。
代表方：归肾丸 或 左归丸。

📊 西医对症用药

  • 递质调节
    SSRIs (如氟西汀、舍曲林)：针对严重经前期情绪暴躁、抑郁（PMDD），可周期性服用（黄体期服用）。
  • 缓解水肿
    螺内酯：针对经前严重的腹胀、水肿、乳房胀痛，具有抗雄及利尿消肿作用。
  • 降泌乳素
    溴隐亭 / 卡麦角林：若抽血确诊高泌乳素血症引起的经前症状。

☯️ 中医分期调理

重点时期：经前期 (黄体期) 治法：疏肝解郁，理气止痛。气血下聚时疏通上半身的“交通”。
代表方：逍遥散。若伴有长痘、口干舌燥、脾气极其火爆等郁热症状，则用 丹栀逍遥散。
经期 (行经期) 治法：活血理气，引血下行。
代表方：柴胡疏肝散 合 四物汤，引血下行，胀痛自消。
经后期 (卵泡期) 治法：柔肝养血。肝体阴而用阳，血足了肝脏才柔和，气才不急躁。
代表方：归脾汤 或 杞菊地黄丸。

📊 西医治疗方案 (对抗雌激素)

  • 急诊止血
    大剂量孕激素 / 诊断性刮宫：急性大出血时，使用黄体酮让内膜全面脱落（药物刮宫），或直接手术清宫快速止血止损。
  • 周期转化
    安宫黄体酮 / 地屈孕酮：每月后半周期连续服用10-14天，人为补充缺乏的孕激素，强行让内膜转化脱落，防止恶变。
  • 长期保护
    LNG-IUS (曼月乐环)：置入子宫内，局部缓慢释放左炔诺孕酮，强烈抑制内膜生长，是治疗内膜增生的首选长效方案。

☯️ 中医“塞流-澄源-复旧”三步法

第一步：出血期 (塞流 - 急则治标) 治法：化瘀止血，固冲摄血。不单纯见血止血，而是“祛瘀生新”。
代表方：失笑散 (五灵脂、蒲黄) 加三七粉，配合 固本止崩汤。强行剥落坏死内膜（中医药物刮宫）。
第二步：止血后 (澄源 - 寻找病因) 治法：温补脾肾，清热或化湿（辨证论治），解决导致崩漏的根本原因，清理体内环境。
第三步：恢复期 (复旧 - 缓则治本) 治法：重建人工周期。
代表方：排卵期前补肾填精 (左归丸)，排卵期后温补脾肾 (右归丸)，恢复自体排卵与正常周期。
